# Cash Counter - Flutter App

**Cash Counter** is a Flutter app designed to help users quickly count cash, track denominations, and manage daily transactions. Perfect for shopkeepers, small businesses, and cash handlers, it offers an intuitive UI, fast performance, and accurate cash management features.

---

## Features

- Count cash quickly and accurately.
- Track different denominations.
- Manage daily cash transactions.
- Simple and user-friendly interface.
- Fast performance and lightweight.
- Export or save cash summaries (optional).
